
## 📊 PROJECT REPORT
This project analyzes supermarket transaction data from a multi-year dataset provided by Commonwealth Bank. Using **Microsoft Excel**, I performed data cleaning, feature engineering, pivot table analysis, and created an interactive dashboard to uncover key business insights.

**Goal:** Transform raw transaction data into actionable business intelligence for optimizing revenue, customer experience, and operations.

**Tools Used:** Microsoft Excel (Pivot Tables, Charts, Dashboard Layout)

---

## 🧾 Introduction
The dataset includes detailed supermarket transactions across multiple years, stores, customer types, payment methods, and products. This analysis focuses on identifying trends in sales performance, customer behavior, payment preferences, and seasonal patterns.

---

## 🛠 Methodology
- **Data Cleaning:** Removed irrelevant columns and handled inconsistencies.
- **Feature Engineering:**
  - Created `Quantity Ordered` = `Total Amount / Unit Price`
  - Extracted `Year` and `Month` from transaction timestamps (e.g., 22-04-2022 09:00)
- **Analysis:** Used Pivot Tables for aggregation and summarization.
- **Visualization:** Built a comprehensive dashboard with multiple chart types for clear pattern recognition.

## 💡 Key Findings Summary
- **Payment Trends:** Contactless is king; credit card usage needs boosting.
- **Yearly Performance:** Highest volume in 2020, but average spend dipped slightly.
- **Customer Segmentation:** Non-members drive the majority of revenue and transactions.
- **Loyalty Gap:** Gold members show lowest engagement.
- **Seasonality:** Strong summer/festive peaks; February slump.
- **Product Focus:** Staples like waffles, eggs, and proteins dominate both volume and revenue.

---

## 💡 Insights & Recommendations
- **Payment Methods:** Invest in faster contactless infrastructure and run campaigns to increase credit card adoption (e.g., bank partnerships or rewards).
- **Customer Loyalty:** Develop targeted strategies to convert non-members into members and re-engage Gold members with personalized offers.
- **Seasonal Strategy:** Launch aggressive promotions in February. Capitalize on peak months (August, December) with bundle deals.
- **Inventory & Store Optimization:** Prioritize stocking top 10 products and share best practices from top-performing stores.
- **Overall:** Focus on non-member experience while strengthening the loyalty program to increase lifetime value.
